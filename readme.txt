== Description ==
Google Recaptcha is a way of spam protecting your website from bots but making it easy for humans to submit the form by simply recreating a checkbox for users to click. To use Google NoCaptcha ReCaptcha for your website you must first register your domain with the ReCaptcha API and get the site API key and the site secret API key.
<br />
Vietnamese: Google Recaptcha là phương pháp bảo vệ website của bạn khỏi những chương trình tự động nhập liệu vào website của bạn bằng cách sử dụng Google NoCaptcha ReCaptcha. Bạn cần <a href="https://www.google.com/recaptcha/admin" target="_blank">đăng ký một API</a> để sử dụng tại đây (miễn phí).


== Installation ==



1. Upload nhanweb_google_recaptcha's folder to the `/wp-content/plugins/nhanweb_google_recaptcha` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
